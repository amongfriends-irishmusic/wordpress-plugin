=== Among Friends Wordpress-Plugin ===

Dieses Plugin implementiert verschiedene Details von AF-WP.


== Installation ==

Eigentlich braucht man nur die Dateien zu kopieren. Es ist zu beachten, dass dieses Plugin zur gemeinsamen Verwendung mit dem Theme `af-gutenberg` gedacht ist.
